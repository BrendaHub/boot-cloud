# netty-websocket

#### 项目介绍
基于netty webscoket实现的网页版即时通讯聊天，点对点，一对多聊天

#### 软件架构
- netty
- springboot


#### 说明
目前netty服务端实现类点对点发消息，群发消息。初学netty练手demo
服务端和客户端可以分开运行。后续集成离线消息发送和接收
